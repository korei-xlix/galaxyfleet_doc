# ESFG：護衛艦
**[Escort Frigate]**  
機動兵器からの対空防御、揚陸してきた歩兵の排除をおこなう、防御専用の護衛艦です。巡航艦がベースとなってます。  
直接艦船を攻撃する能力は低いですが、巡航艦より装甲と機動性能に優れ、接近する敵艦船から艦隊を防御します。  
しかし、コストがやや高くなっています。  
　　主な武器：対空砲  


## ESFG-STD：護衛艦
**[Escort Frigate]**  
対空戦闘力と電子性能に優れ、艦隊の防御全般をおこなう戦闘艦です。  
電子戦装備を搭載し、索敵や電子妨害、電子妨害への対抗といった電子戦を得意とします。  
ロケットランチャーと対空砲を搭載し、近距離の敵艦や航空機の迎撃がおこなえます。ミサイル迎撃能力があります。  
また、艦内防衛用に陸戦歩兵を運用でき、味方艦に移乗してきた敵の歩兵を排除する能力があります。  
万能な分コストがかかります。

# ESCV：防護艦
**[Escort Corvet]**  
脅威の早期発見や、電子戦や機動戦による妨害や攪乱、電子妨害からの防御をおこなう、防御専用の護衛艦です。駆逐艦がベースとなってます。  
直接艦船を攻撃する能力は低いですが、駆逐艦より装甲と電子性能に優れ、電子戦により艦隊防御をおこないます。  
コストが低い分、護衛艦より性能が低いです。  
　　主な武器：対空砲

# ESEM：護衛母艦
**[Escort Mother Ship]**  
制空用や哨戒用の艦載機で艦隊の防空、対潜哨戒、周辺索敵をおこなう母艦型の護衛艦です。汎用母艦がベースとなってます。  
空母よりも低コストで制空戦闘機を扱えるところにメリットがあり、更に電子性能も高いのが特徴です。  
ただ、元々は周辺宙域の哨戒や、商船護衛用のため移動速度が遅いです。  
　　主な武器：対空砲  


## ESEM-STD：護衛母艦
**[Escort Mother Ship]**  
制空用の航空機で艦隊防空ができる母艦型の護衛艦です。  
制空、偵察、対潜、電子戦用の艦載機が運用できます。  
対空砲しか搭載できず、単艦の戦闘力は貧弱です。  
搭載機の搭載量が調整され、要撃機、戦闘機、重戦闘機が増え、それ以外の搭載機の搭載量が減ります。  
飛行甲板が損傷すると艦載機の離着陸が発進できなくなります。
