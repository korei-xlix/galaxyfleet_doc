# BSDS：駆逐艦
**[Destroyer]**  
高速で小回りがきく小型の戦列艦です。  
近距離での魚雷攻撃、爆雷での対潜攻撃を得意とします。  
　　主な武器：小型主砲、魚雷、爆雷、対空砲

**[Destroyer]**  
標準性能の駆逐艦です。小回りが効いて、魚雷戦や対潜が得意です。

# BSCR：巡航艦
**[Cruiser]**  
中型で長距離の航行に向いてる戦列艦です。  
中型の火砲による砲戦が得意で、機動性が高く、索敵にも向いています。  
艦載機として偵察機を搭載できます。  
　　主な武器：中型主砲、小型主砲、偵察機、対空砲

**[Cruiser]**

接近戦では駆逐艦、長距離戦では戦艦に見劣り、戦列艦としては凡庸な戦闘性能です。
