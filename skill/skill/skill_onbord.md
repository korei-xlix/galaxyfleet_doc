  [目次へ戻る](#aMokuji)  
  

<h2>揚陸貨物庫 / </h2>
戦術マップの降下地点に止まっている時、資源や資材の出し入れができます。  
貨物ペイロードが 80 割り当てられます。  
追加貨物庫が搭載できます。  
桟橋や仮桟橋がなくても出し入れができます。  
　　追加機能：上陸  
  

<h2>桟橋付き貨物庫 / </h2>
戦略マップで桟橋か、仮桟橋に止まっている時、資源や資材の出し入れができます。  
貨物ペイロードが 120 割り当てられます。  
追加貨物庫が搭載できます。  
輸送艇も1隻搭載できます。  
桟橋か仮桟橋がないと出し入れはできません。  
  

<h2>大型貨物庫 / </h2>
戦略マップで桟橋か、仮桟橋に止まっている時、資源や資材の出し入れができます。  
貨物ペイロードが 200 割り当てられます。  
追加貨物庫が搭載できます。  
桟橋か仮桟橋がないと出し入れはできません。  
  





<h1 id="aAircraftEquiped">航空機搭載 / Aircraft Equiped</h1>  
艦船に内蔵された飛行甲板と格納庫のセットです。  
  
  [目次へ戻る](#aMokuji)  
  

<h2>航空甲板 / </h2>
航空母艦用の飛行甲板と格納庫です。艦載機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
　　搭載可能：空母航空機、重戦闘機、歩兵支援機  
　　追加機能：空母直掩、航空打撃、航空侵入  
  

<h2>装甲甲板 / </h2>
航空母艦用の装甲飛行甲板と耐爆格納庫です。艦載機の発着、格納、修理、補充ができます。  
被弾ダメージを10％カットし、損傷しても艦載機の発着ができます。  
誘爆対策で爆弾を扱う艦載機が搭載できません。  
　　搭載可能：高速爆撃機、戦闘爆撃機以外の空母航空機  
　　　　　　　重戦闘機、歩兵支援機  
　　追加機能：空母直掩、航空打撃、航空侵入  
　　発動効果：硬い飛行甲板  
  

<h2>強化甲板 / </h2>
航空母艦用の強化飛行甲板と格納庫です。艦載機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
攻撃向けの機能が追加されています。  
　　搭載可能：空母航空機、重戦闘機、歩兵支援機  
　　追加機能：空母直掩、航空打撃、航空侵入、航空反転、航空襲撃  
  

<h2>大型甲板 / </h2>
大型空母用の飛行甲板と格納庫です。大型航空機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
　　搭載可能：大型航空機  
　　追加機能：空母直掩、航空打撃、航空侵入  
  

<h2>汎用航空甲板 / </h2>
戦艦、巡洋艦用の小型航空甲板、格納庫のセットです。艦載機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
　　搭載可能：汎用航空機  
　　追加機能：空母直掩、航空打撃、航空侵入  
  

<h2>戦闘甲板 / </h2>
戦闘空母用の飛行甲板と格納庫です。艦載機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
甲板を切り替えることで砲撃モードに移行することができるが、砲撃モード中は艦載機の発着ができません。  
　　搭載可能：空母航空機、重戦闘機、歩兵支援機  
　　追加機能：空母直掩、航空打撃、航空侵入、甲板切替  
  

<h2>巡洋艦用射出設備 / </h2>
巡洋艦用の小型射出カタパルト、回収クレーン、格納庫のセットです。艦載機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
　　搭載可能：汎用戦闘機、汎用偵察機  
  

<h2>戦艦用射出設備 / </h2>
戦艦用の大型射出カタパルト、回収クレーン、格納庫のセットです。艦載機の発着、格納、修理、補充ができます。  
甲板が被弾すると、艦載機の発着ができなくなります。  
　　搭載可能：汎用戦闘機、汎用爆撃機、汎用戦爆機、汎用偵察機  
  

<h2>多段式甲板 / Multi Stage Deck</h2>
多数の航空機を運用できるよう複数の飛行甲板が搭載されます。  
搭載機の搭載数が最大値の 30％ 増えます。  
装甲が手薄になることで艦体の耐久性が落ちるため、被弾ダメージが5％上昇します。  
  

<h2>要撃基地 / Interceptor Base</h2>
要撃空母用の甲板構造です。  
要撃機、戦闘機、重戦闘機の搭載機の搭載数が最大値の 40％ 増えます。  
そのかわり要撃機、戦闘機、重戦闘機以外の搭載機の搭載数が最大値の 20％ 減ります。  
　　追加機能：航空要撃、航空防御  
  

<h2>戦略基地 / Interceptor Base</h2>
戦略空母用の甲板構造です。  
搭載機の搭載数が最大値の 30％ 増えます。  
追加で陸戦爆撃機が運用できるようになります。  
装甲が手薄になることで艦体の耐久性が落ちるため、被弾ダメージが5％上昇します。  
　　追加機能：航空戦略、航空一斉  
  

<h2>制宙艦 / Space Control</h2>
制宙艦の甲板構造です。  
偵察機、対潜攻撃機、対潜哨戒機の搭載機数が最大値の 40％ 増えます。  
そのかわり要撃機、戦闘機、重戦闘機以外の搭載機の搭載数が最大値の 20％ 減ります。  
また潜航艦への索敵が 30％ 向上します。  
  






<h1 id="aInfantryEquiped">歩兵搭載 / Infantry Equiped</h1>

<h2>歩兵発着設備 / </h2>
歩兵用の発着設備、格納庫です。歩兵の発着、格納、修理、補充ができます。  
また陸戦歩兵を搭載している場合、艦船への移乗ができます。  
　　搭載可能：機動歩兵、陸戦歩兵  
　　追加機能：歩兵移乗、上陸  
  

<h2>機兵発着設備 / </h2>
機動歩兵用の発着設備、格納庫です。機動歩兵の発着、格納、修理、補充ができます。  
　　搭載可能：機動歩兵  
  







<h1 id="aBoatEquiped">装載艇搭載 / Boat Equiped</h1>  
  
  [目次へ戻る](#aMokuji)  
  

<h2>装載艇発着設備 / </h2>
装載艇用の発着設備、格納庫です。装載艇の発着、格納、修理ができます。  
装載艇が 2 隻搭載できます。  
　　搭載可能：装載艇  
  





<h1 id="aLandUnitEquiped">陸戦搭載 / Land Unit Equiped</h1>  
  
  [目次へ戻る](#aMokuji)  
  

<h2>陸戦揚陸設備 / </h2>
揚陸艦用の発着設備、格納庫です。陸戦兵器の発着、格納、修理ができます。  
陸戦歩兵、戦車、陸戦攻撃機が搭載できます。ただし、陸戦攻撃機は運搬のみで、搭載艦からの発艦はできません。  
また、輸送艇、揚陸艦が 2 隻搭載できます。  
　　追加機能：上陸  
　　搭載可能：陸戦歩兵、戦車、陸戦攻撃機（運搬のみ）  
　　　　　　　輸送艇、揚陸艦  
  

<h2>戦車専用 / For Tanks Onlyn</h2>
戦車しか搭載できなくなります。  
その代わり、戦車の搭載量が 40％ 増えます。

<h2>格闘 / Fighting</h2>  
接近戦用武器を装備している場合に発動します。  
接近戦用武器でのダメージが 50％、命中率が 20％ 上昇します。  
また近接レンジにいる敵の攻撃の回避率が 30％ 上昇します。  
  





航空甲板 / Flight Deck
空母用飛行甲板が使えます。塔載にコストがかかりません。  


装甲甲板 / Armor Deck
重装甲飛行甲板が使えます。塔載にコストがかかりません。  


強化甲板 / Reinforced Deck
強化飛行甲板が使えます。塔載にコストがかかりません。  







## 巨大甲板
**[Huge Deck]**  
大型航空機を運用するための甲板構造です。  
大型航空機の搭載数が最大値の 50％ 増えます。  


## 高速甲板
**[Speed Deck]**  
発進装置の強化だけでなく、誘導路、格納庫も改良し、搭載機を可能なかぎり高速で発進させることができます。  
ただし、搭載機の搭載数が最大値の 30％ 減ります。  


## 爆撃基地
**[Bomber Base]**  
超大型空母用の甲板構造です。  
攻撃機や爆撃機の搭載数が最大値の 40％ 増えます。  
その代わり、戦闘機、機動歩兵の搭載数が最大値の 20％ 減ります。  




## 舟艇内蔵
**[Stores Boats]**  
舟艇の搭載量が2倍になります。  






## 航空化
**[Aviation]**  
コストなしで汎用高性能飛行甲板が搭載できるようになります。  





## ドローンオーナー
**[Drone Owner]**  
コストなしでドローン用設備が搭載できるようになります。
