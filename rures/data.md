# ユーザ管理
ユーザデータを含めたゲームのプレイデータは、基本的にローカル環境側のみでロード、セーブを行う。処理負荷やセキュリティを考慮し、処理サーバに送ったり、ブラウザのストレージに保管したりはしない。なおセーブデータにはチェックサムのような識別データを付加し、外から改ざんできないようにする。パスワードも暗号化する（ロジックはコード側で準備済み）。

|属性|名前|備考|
|:--|:--|:--|:--|
|ユーザID|ログイン時に使うID。|半角英数字のみ|
|パスワード|ログイン時に使うパスワード。|半角英数字一部システム文字|
|司令官名|ゲーム内で表示する司令官の名前。|半角全角英数字|

> **将来的なシステム**
> 将来的にはユーザ間を支援する友軍システムや、スコアを表示するようなシステムを作りたい。
> こちらは個々ではなく、総司令部が用意するシステムになる。

# 兵器の基礎構造
**[Basic Structure of Unit]**  
兵器は、基礎素材、姿勢制御装置、電算装置、エンジンで構成されます。それぞれに素材の種類があり、その組み合わせで基本ステータスが変わります。  




<a id="iBaseStatus"></a>
# 兵器の基礎ステータス
**[Base Status of Unit]**  
兵器の基礎ステータスは、耐久性能(DF)、機動性能(MV)、電子性能(EL)、動力性能(GN)、戦闘性能(BT)の５つに分かれます。それぞれの基本ステータス値は基礎構造で選択された素材の組み合わせによって変化します。基礎素材は耐久性能、姿勢制御装置は機動性能、電算装置は電子性能、エンジンは動力性能に影響します。艦船と装載艇以外は、耐久性能(DR)、動力性能(GN)の変わりに生存性能(SV)、稼働性能(OP)が適用されます。ただし、経験能力なので基礎構造はありません。  
さらに各基礎ステータスには、付属ステータスがあります。基礎値が上昇すると付属ステータスも上昇します。運(LK)は隠しステータスです。  


## 艦船のステータス
* DR：耐久性能　[Durability]  
  兵器の耐久度を示す基礎値です。  
  数値が高いほど生存性が高く、荷物が積めて、頑丈になります。  
  ただし、目立ちやすくなり索敵にひっかかりやすくなります。  
  
|数値が大きい時の効果    |  |
|:--|:--|
|生存性が高い            |LP(+)     |
|荷物持てる              |DLC(+)    |
|頑丈                    |DAM(+)    |
|目立ちやすくなる        |DVR(-)    |
|攻撃が遅くなる          |[MMB](-)  |
|移動が鈍くなる          |[MMV](-)  |
|伝達力が低くなる        |[PTP](-)  |

* MB：機動性能　[Mobility]  
  兵器の加速度や旋回性能を示す基礎値です。  
  数値が高いほど艦船のスピードが速く、攻撃の際早く攻撃できます。  
  しかし移動距離が短くなります。  
  
|数値が大きい時の効果    |  |
|:--|:--|
|大きいほど素早い        |MAV(+)    |
|攻撃順が早い            |MMB(+)    |

* EE：電子性能　[Electronic]  
  兵器の演算能力、処理能力を示す基礎値です。  
  数値が高いほど電子処理に優れます。  
  
|数値が大きい時の効果    |  |
|:--|:--|

## DR：耐久性能
* LP：ライフ値　[Life Point]  
  兵器の生存可能値です。0になると撃沈します。  
  回復するには港湾や船渠、補修機能のある艦船で修復します。  
　【DR】~2 * ≪DR素材補正≫  
　強化装備：フレーム強化パーツ  

* DAM：装甲　[Armor]  
  兵器の防御値です。  
　(【DR】 * ≪DR素材補正≫ ) - ＜MB素材補正＞  
　強化装備：追加装甲  

* DVR：視認　[Visual Recognition]  
  外からの兵器識別値です。  
  数値が"低い"と索敵から回避しやすくなります。耐久と装備の大きさに影響します。  
　(【DAM】 * ＜DR素材補正＞ ) - (【MB】 * ≪MB素材補正≫) + ＜EE素材補正＞ + 装備CAP(計)  
　強化装備：レーダ反射板  

* DLC：荷物容量　[Luggage Capacity]  
  荷物の収納値です。HPが高くなると広くなります。  
　〖最大LP】 - ＜MB素材補正＞ - ＜EE素材補正＞ - 装備CAP(計)  
　強化装備：空間拡張キット  


## MB：機動性能
* MAV：回避　[Avoidance]   
  攻撃からの回避力です。  
　【MB】 * ≪MB素材補正≫ + ≪EE素材補正≫ - ＜PW素材補正＞  
　強化装備：姿勢制御補助装置  

* MMB：機動力　[Mobility]  
  攻撃順番にかかわります。  
　【MB】~2 + ≪MB素材補正≫ + ≪EE素材補正≫ - ＜DR素材補正＞   
　強化装備：加速補助装置  

* MMV：移動距離　[Moving Distance]  
  戦場の移動力です。  
　【MB】~2 - ＜MB素材補正＞ - ＜DR素材補正＞ + ≪PW素材補正≫  
　強化装備：航行補助装置  


## EE：電子性能
* ECA：電算力　[Computer Accuracy]
  電子戦を仕掛けたり、対抗する力です。  
  【EE】^2 + ≪EE素材補正≫
　強化装備：増設演算装置  

* EAM：捕捉　[AIM]  
  武器の照準を支援し命中率をあげたり、ミサイル迎撃力に関わります。  
　【EE】 - ＜PW素材補正＞
　強化装備：火器管制補助装置  

* EDT：探知　[Detection]  
  敵を探したり、隠し要素や、次元に隠れている潜航艦を発見する力です。  
　【EE】 + ≪MB素材補正≫ - ＜PW素材補正＞
　強化装備：レーダ補助装置  


## PW：動力性能
* SP：シールド値　[Shield Point]  
  シールド値です。LPよりも先にダメージを受けます。  
  とくに資材を使わずにシールドジェネレータにより自然回復します。  
  0になるとシールドダウンし、次回からLPにダメージを受けるようになります。  
  XXXXXX のあまりを加算補正としてSPに回せます。  
  【PW】~2 + ≪PW素材補正≫  
　強化装備：シールド増幅装置  

* PEC：装備容量値　[Equipment Capacity]  
  最大装備CAP値です。装備ENPの合計がこれを上回る装備は搭載できません。  
　【PW】~2 + ≪PW素材補正≫ - ＜EE素材補正＞  
　強化装備：動力増幅装置  

* PTP：伝達力　[Transmission Power]  
  エネルギーを効率よく伝達する力です。  
  リチャージ速度の上昇、エネルギーの消耗が抑えられます。  
  【PW】~2 + ≪EE素材補正≫ - ＜DR素材補正＞  
　強化装備：伝達補助装置  

* PAR：AP回復力　[AP Resilience]  
  消耗したAPの回復力です。 
  【PW】 + ≪PW素材補正≫  
　強化装備：燃料フィルタ
